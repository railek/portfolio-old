# 💾 Railek

## 🛒 Requirements

Go to the [official Node.js website](https://nodejs.org/) and download the installer for your approriate operating systems.
Do the same for [git](https://git-scm.com/).

## 🧷 Install

```shell
git clone git@github.com/railek/railek.com.git
cd railek.com
npm install
```

## 🖥 Development

```shell
npm run dev
```
